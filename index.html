Mythical Twitter Bot

"Python based bot"


Mythical twitter bot allows you to create one bot easily. All you need is one twitter account that's it. This program will require username and password for authentication and then after that it will do your work like making tweet every 30 min at any particular page or to any specific person.

And interesting part is that it works without using Twitter API. Yes, it works all by scraping and gathering information.

git clone https://github.com/shashank-sharma/mythical-bot-twitter
